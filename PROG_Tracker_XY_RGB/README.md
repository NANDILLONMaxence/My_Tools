# PROG Tracker XY RGB

## Instructions d'utilisation :

1. **Installation des dépendances :**
   - Assurez-vous d'avoir Python installé sur votre système.
   - Installez les bibliothèques nécessaires en exécutant la commande suivante dans votre terminal :
     ```
     pip install threaded
     ```
	 ```
	 pip install Everything-Tkinter
	 ```
	 ```
	 pip install pillow
	 ```
	 
2. **Exécution du script :**
   - Assurez-vous d'avoir Python installé sur votre système.
   - Exécutez le script en utilisant la commande suivante :
     ```
     python nom_du_script.py
     ```
	 ou en exécutant le `.exe`

2. **Interface Utilisateur :**
   - L'étiquette "X: 0, Y: 0" affiche les coordonnées actuelles du curseur de la souris.
   - L'étiquette "RGB: N/A" affiche la couleur RGB du pixel sous le curseur (si activé).
   - Cochez la case "Capture Coordinates" pour activer la capture des coordonnées du curseur.
   - Cochez la case "Capture RGB" pour activer la capture de la couleur RGB du pixel sous le curseur.

3. **Démarrage de la Capture :**
   - Cliquez sur le bouton "Start" pour démarrer la mise à jour en temps réel des informations du curseur.
   - Si aucune option n'est sélectionnée, un message d'erreur sera affiché en rouge.

4. **Arrêt de la Capture :**
   - Cliquez sur le bouton "Stop" pour arrêter la mise à jour en temps réel.

5. **Avertissement :**
   - Assurez-vous d'avoir les autorisations nécessaires pour utiliser les fonctionnalités de capture du curseur.
   - La capture de la couleur RGB peut être activée ou désactivée à tout moment.
